## About

Math++ is a comprehensive web application designed to help students, programmers, and enthusiasts master number systems and arithmetic operations across different bases. Built entirely with vanilla HTML, CSS, and JavaScript.

**Live App:** [https://math-plus-plus.netlify.app](https://math-plus-plus.netlify.app)

---

## Features

- **Number System Explorers** - Explore Decimal, Binary, Octal, and Hexadecimal systems with instant cross-conversions
- **Universal Converter** - Convert between any number systems with real-time validation
- **Intensive Arithmetic** - Binary, Hexadecimal, Octal, and mixed-system calculations
- **Interactive Exercises** - Practice questions with hints and detailed explanations
- **Mastery Quiz** - 5-question quiz with scoring and performance feedback
- **Progress Tracking** - LocalStorage saves achievements and unlocks badges

---

## Built With

| Technology | Purpose |
|------------|---------|

| JavaScript (Vanilla) | All logic, conversions, arithmetic, localStorage |
