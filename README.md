## 📖 About

Math++ is a comprehensive web application designed to help students, programmers, and enthusiasts master number systems and arithmetic operations across different bases. Built entirely with vanilla HTML, CSS, and JavaScript.

**Live App:** [https://math-plus-plus.netlify.app](https://math-plus-plus.netlify.app)

---

## ✨ Features

### 🔢 Number System Explorers
- Explore Decimal, Binary, Octal, and Hexadecimal systems
- See instant conversions across all bases simultaneously

### 🔄 Universal Converter
- Convert between any number systems (Binary, Octal, Decimal, Hexadecimal)
- Real-time validation and error handling

### 🧮 Intensive Arithmetic
- Binary addition, subtraction, multiplication
- Hexadecimal and Octal arithmetic
- Mixed system calculator (operate across different bases)

### 📚 Interactive Exercises
- 5 practice questions with increasing difficulty
- Detailed explanations for correct answers
- Hints to guide learning

### 🎯 Mastery Quiz
- 5-question quiz to test your knowledge
- Instant scoring and performance feedback
- Progress tracking across attempts

### 📊 Progress Tracking
- localStorage saves your achievements
- Unlock badges: Converter Master, Binary Expert, Hex Pro, Quiz Master
- Track problems solved and accuracy rate
